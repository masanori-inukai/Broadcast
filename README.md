Broadcast
====

RTMPを使用してLIVE配信を行うためのアプリです。
カメラからの画像と、コントロールセンターの画面収録からのBroadcastに対応しています。

## 

## Requirement

1. Swift 4.1
2. iOS 11.0 以上 (カメラだけならiOS 8.0)

## Install

1. bundle install --path vendor/bundle
2. bundle exec pod setup
3. bundle exec pod install
